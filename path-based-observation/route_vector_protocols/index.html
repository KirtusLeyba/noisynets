Path Vector Routing

Routing protocols, such as the Border Gateway Protocol (BGP), where paths between regions of IP space are advertised by routers to propogate reachability information. Public data for BGP is collected by route collectors, which gather BGP announcements from various vantage points (VPs). Datasets containing public routing information include paths through Autonomous Systems (ASes).
